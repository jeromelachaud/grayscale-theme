What is the study topic of this paper?

In this paper, how drivers perform visual sampling on an in-car interface, specifically when searching through a large number of unordered visual items spread across multiple screens is measured

Grid layout vs. List layout

Kujala and Saariluoma (2011) found higher individual in-car glance durations by increasing the number of items per screen as well as increased glance durations for a grid-style menu layout compared to a list layout.
